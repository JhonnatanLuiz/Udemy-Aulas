Visão geral do Bootstrap

Bootstrap é o toolkit front-end mais popular do mundo para construir interfaces responsivas com velocidade. Ele oferece uma combinação de layout flexível, componentes prontos e utilitários.

Pilares do framework

  • Grid responsivo baseado em flexbox
  • Componentes estilizados e acessíveis
  • Utilitários para ajustes rápidos
  • Plugins JavaScript para interatividade

Instalação e setup

Fundamentos

Bootstrap pode ser utilizado via CDN, pacotes npm ou importação via bundlers modernos. Use a abordagem adequada ao seu projeto.

CDN rápido

Ideal para protótipos e estudos. Inclua os links abaixo na sua página.

<!-- CSS -->
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">

<!-- JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Instalação via npm

Perfeito para projetos escaláveis e integração com ferramentas de build.

npm install bootstrap@5.3.3

// Importando no seu bundle JS
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';

// Ou importando apenas componentes via Sass
Dica: sempre adicione a meta tag viewport para garantir que o layout responda bem em dispositivos móveis.

Utilitários para produtividade

Turbo

Classes utilitárias permitem ajustes sem escrever CSS adicional. Memorize as mais frequentes.

Espaçamentos

Use padrões como p-*, m-* e suas variações direcionais.

p-3 px-4 pt-lg-5 m-2 mt-md-4

Tipografia e cores

Combine classes de texto e background para realçar informações.

fw-bold text-uppercase text-success bg-warning-subtle

Clique para visualizar rapidamente onde cada utilitário está aplicado na página.

Customização & theming

Avançado

Personalize cores, tipografia e componentes utilizando variáveis CSS ou compilação Sass.

Variáveis CSS

:root {
  --bs-primary: #6f42c1;
  --bs-body-font-family: 'Inter', sans-serif;
}

.btn-custom {
  --bs-btn-bg: var(--bs-primary);
  --bs-btn-border-color: var(--bs-primary);
  --bs-btn-hover-bg: #5a32a3;
  --bs-btn-color: #fff;
}

Sass (opcional)

// Instale Sass
npm install sass

// Importe variáveis
@import "node_modules/bootstrap/scss/bootstrap";

$primary: #ff6f61;
$enable-rounded: false;

// Compile com: sass input.scss output.css

Objetivo: Criar um card com imagem, título, descrição e 2 botões.

Requisitos:

  • Inclua um badge "Novo"

    <a href="#" class="btn btn-outline-secondary">Detalhes</a>

</div>

Objetivo: Criar uma barra de navegação completa com menu dropdown e campo de busca.

Requisitos:

  • Navbar responsiva com botão collapse
  • Menu dropdown funcional
  • Campo de busca alinhado à direita
<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Logo</a>
    <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav me-auto">
        <li class="nav-item dropdown">
          <a class="nav-link dropdown-toggle" href="#" data-bs-toggle="dropdown">Menu</a>
          <ul class="dropdown-menu">
            <li><a class="dropdown-item" href="#">Opção 1</a></li>
            <li><a class="dropdown-item" href="#">Opção 2</a></li>
          </ul>
        </li>
      </ul>
      <form class="d-flex">
        <input class="form-control me-2" type="search" placeholder="Buscar">
        <button class="btn btn-outline-primary" type="submit">Buscar</button>
      </form>
    </div>
  </div>
</nav>

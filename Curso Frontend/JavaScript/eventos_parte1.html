Resumo dos Conceitos Aprendidos

  • addEventListener(): Método moderno para adicionar eventos
  • Objeto Event: Contém informações sobre o evento ocorrido
  • preventDefault(): Cancela o comportamento padrão do evento
  • Event Bubbling: Eventos se propagam do elemento filho para o pai
  • Event Capturing: Eventos se propagam do elemento pai para o filho
  • Tipos de Eventos: Mouse, teclado, formulário, janela, etc.
